## Featured Project

### [Drink Daily](https://drink-daily.vercel.app/en)

A modern hydration tracking web app designed to be **free, registration-free, and genuinely useful from the first visit**.

**What it offers:**
- daily hydration tracking
- a frictionless experience with **no sign-up required**
- a full-featured product feel rather than a barebones demo
- clean UX designed for everyday use
